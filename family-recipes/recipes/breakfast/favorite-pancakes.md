# Favorite Pancakes

## Measured Ingredients

### Favorite Pancakes

| Amount | Ingredient | Notes |
|---:|---|---|
| 1 | Egg |  |
| 1 cup | Buttermilk |  |
| 2 Tbsp | Cooking oil |  |
| 1 cup | Flour |  |
| 1 Tbsp | Sugar |  |
| 1 tsp | Baking powder |  |
| 1/2 tsp | Salt |  |
| 1/2 tsp | Baking soda |  |

## Instructions

| Step | Action |
|---:|---|
| 1 | Beat the egg. |
| 2 | Add the remaining ingredients in the order listed. |
| 3 | Beat with a whisk until smooth. |
| 4 | Pour batter slowly onto a lightly greased griddle that has been preheated to a moderately high temperature. |
| 5 | Turn pancakes as soon as they are puffed and full of bubbles, but before the bubbles break. |
| 6 | Cook the other side until golden brown. |
