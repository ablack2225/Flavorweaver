# 10-Minute Spaghetti Sauce

> A preserved family spaghetti sauce recipe made with ground beef, onion, tomato sauce, tomato paste, garlic powder, sweet basil, oregano, marjoram, brown sugar, and black pepper.

## Measured Ingredients

### 10-Minute Spaghetti Sauce

| Amount | Ingredient | Notes |
|---:|---|---|
| 1/2-1 lb | Ground beef |  |
| 1/2 medium | Onion | Chopped |
| 2 (8 oz) cans | Tomato sauce |  |
| 1 (6 oz) can | Tomato paste |  |
| 1/2 tsp | Garlic powder |  |
| 1 1/2 tsp | Sweet basil |  |
| 1 tsp | Oregano |  |
| 1/4 tsp | Marjoram |  |
| 1 1/2 tsp | Brown sugar |  |
| To taste | Black pepper |  |
| 1/2-1 cup | Water | Add until desired consistency is reached |

## Instructions

| Step | Action |
|---:|---|
| 1 | Brown ground beef and onion. |
| 2 | Drain off excess fat. |
| 3 | Add all remaining ingredients except water and mix well. |
| 4 | Add water until desired consistency is reached. |
| 5 | Simmer about 10 minutes. |
| 6 | Serve over your favorite pasta. |
