# Eggnog

> A preserved family holiday eggnog made with eggs, milk, sugar, nutmeg, cinnamon, and allspice, chilled before serving.

## Measured Ingredients

### Eggnog

| Amount | Ingredient | Notes |
|---:|---|---|
| 1/2 cup | Sugar |  |
| 1/4 tsp | Nutmeg |  |
| 1/4 tsp | Cinnamon |  |
| 1/8 tsp | Allspice |  |
| 4 | Eggs | Separated |
| 3 1/2 cups | Cold milk |  |
| Optional | Bourbon whiskey | Add a little to each glass, if desired |

## Instructions

| Step | Action |
|---:|---|
| 1 | Combine sugar, nutmeg, cinnamon, and allspice in a small bowl. |
| 2 | In a large bowl, beat egg whites until soft peaks form. |
| 3 | Add half of the sugar mixture to the egg whites and beat until stiff peaks form. |
| 4 | In a small bowl, beat egg yolks until thick and lemon colored. |
| 5 | Add the rest of the sugar mixture to the egg yolks and mix well. |
| 6 | Fold the yolk mixture into the whites, blending well. |
| 7 | Stir in cold milk and pour into a container. |
| 8 | Chill several hours before serving. |
| 9 | Shake well before pouring into glasses. |
| 10 | Optional: add a little bourbon whiskey to each glass. |

## Notes

Original family note: adding a little bourbon whiskey to each glass really mellows out the flavor.

This preserved recipe contains raw eggs. Use pasteurized eggs if preferred.

Servings, chill time, and total time are estimated from the ingredient amounts and the instruction to chill several hours before serving. The original recipe page listed prep time as 10 minutes and left servings blank.
