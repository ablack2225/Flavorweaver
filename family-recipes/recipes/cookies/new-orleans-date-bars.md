# New Orleans Date Bars

> A preserved family date bar recipe made with brown sugar, chopped dates, chopped nuts, cinnamon, and a confectioners' sugar finish.

## Measured Ingredients

### New Orleans Date Bars

| Amount | Ingredient | Notes |
|---:|---|---|
| 6 Tbsp | Margarine |  |
| 2 | Eggs |  |
| 1 1/2 tsp | Baking powder |  |
| 1/4 tsp | Salt |  |
| 3/4 cup | Brown sugar |  |
| 1 1/2 cups | Flour |  |
| 1/4 tsp | Cinnamon |  |
| 1/4 cup | Milk |  |
| 3/4 cup | Chopped nuts |  |
| 1 (8 oz) package | Chopped dates |  |
| As desired | Confectioners' sugar | For dusting after cooling |

## Instructions

| Step | Action |
|---:|---|
| 1 | Preheat oven to 350°F. |
| 2 | Beat margarine and brown sugar until light and creamy. |
| 3 | Add eggs and beat well. |
| 4 | Combine dry ingredients. |
| 5 | Stir dry ingredients alternately with milk into the sugar mixture. |
| 6 | Stir until all dry ingredients are well mixed into the batter. |
| 7 | Stir in dates and nuts. |
| 8 | Spread batter into a greased 9x9x2-inch baking dish. |
| 9 | Bake for about 35 minutes. |
| 10 | Cool. |
| 11 | Dust with confectioners' sugar. |
| 12 | Cut into bars. |
