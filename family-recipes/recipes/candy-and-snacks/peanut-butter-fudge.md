# Peanut Butter Fudge

> A preserved family microwave peanut butter fudge made with peanut butter chips, peanut butter, marshmallow cream, evaporated milk, sugar, margarine, vanilla, and chopped nuts.

## Measured Ingredients

### Peanut Butter Fudge

| Amount | Ingredient | Notes |
|---:|---|---|
| 3/4 cup | Margarine | Melt first |
| 3 cups | Sugar |  |
| 2/3 cup | Evaporated milk |  |
| 1 package | Reese's peanut butter chips |  |
| 1 (7 oz) jar | Marshmallow cream |  |
| 2 heaping Tbsp | Peanut butter |  |
| 1 cup | Chopped nuts |  |
| 1 tsp | Vanilla |  |

## Instructions

| Step | Action |
|---:|---|
| 1 | In a large microwavable bowl, melt margarine for 1 minute. |
| 2 | Add sugar and evaporated milk. |
| 3 | Microwave for 5 minutes, stirring after 3 minutes and again at 5 minutes. |
| 4 | Microwave an additional 5 1/2 minutes, stirring again after 3 minutes. |
| 5 | Remove from microwave and stir in peanut butter chips until completely melted. |
| 6 | Add peanut butter, marshmallow cream, and vanilla. Stir until well blended. |
| 7 | Pour into a greased 13x9-inch pan. |
| 8 | Cool completely before cutting. |
| 9 | Store in an airtight container. |
