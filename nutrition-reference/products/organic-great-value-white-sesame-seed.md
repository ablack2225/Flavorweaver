## Flavorweaver Calculation Notes

Use this reference when a recipe uses Organic Great Value White Sesame Seed with the same 1/4 tsp / 0.9g nutrition profile.

Because the serving size is very small, the label rounds fat, carbohydrate, and protein to 0g. Calories should still be counted.

Quick conversion from label serving:

```text
1/4 tsp sesame seeds = 5 calories
1 tsp sesame seeds = 4 label servings = 20 calories
1 Tbsp sesame seeds = 3 tsp = 12 label servings = 60 calories
2 Tbsp sesame seeds = 120 calories
```

## Verification Notes

This label uses a very small serving size, which causes macro rounding. For recipe macro precision, sesame seed calories are useful, but fat/carbohydrate/protein may be underreported if relying only on the rounded label panel.

If a recipe uses a larger amount of sesame seeds and tighter macro accuracy is needed, create a household calculated standard by grams or use a USDA-style sesame seed standard alongside this product label.

If using black sesame seeds, toasted sesame seeds, sesame oil, tahini, sesame paste, or another brand, create a separate product reference or mark the calculation as estimated pending label.
