## Verification Notes

This bouillon is sodium-dense, so measure carefully. It contributes very little calorie or macro load, but it can significantly increase sodium totals.

For recipe accuracy, calculate from prepared-cup yield unless Amy uses the dry powder directly in a different amount.

If using boxed chicken broth, homemade broth, Better Than Bouillon, stock concentrate, low-sodium bouillon, bone broth, or another brand, create a separate product reference or mark the calculation as estimated pending label.
