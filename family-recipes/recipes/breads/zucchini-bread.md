# Zucchini Bread

> A preserved family zucchini bread made with grated zucchini, warm spices, chopped nuts, and optional chopped dates.

## Measured Ingredients

### Zucchini Bread

| Amount | Ingredient | Notes |
|---:|---|---|
| 3 | Eggs |  |
| 2 cups | Sugar |  |
| 1 cup | Cooking oil |  |
| 2 cups | Zucchini | Peeled and grated |
| 2 tsp | Vanilla |  |
| 3 cups | Flour |  |
| 1 tsp each | Salt, baking soda, and baking powder |  |
| 1 tsp each | Ground cloves and cinnamon |  |
| 1/2 tsp | Ginger |  |
| 1 cup | Chopped nuts |  |
| 1 package | Chopped dates | Optional |

## Instructions

| Step | Action |
|---:|---|
| 1 | Preheat oven to 325°F. |
| 2 | Beat together eggs, sugar, oil, and vanilla. |
| 3 | Add the grated zucchini. |
| 4 | Mix the dry ingredients together. |
| 5 | Add the dry ingredients to the egg mixture, mixing until blended well. |
| 6 | Add the chopped nuts and dates. |
| 7 | Pour into a large greased loaf pan or 2 small loaf pans. |
| 8 | Bake for about 1 hour, or until completely done through. Small loaves may cook faster. |
| 9 | Allow to cool in the pan about 10 minutes. |
| 10 | Remove from pan and cool completely before wrapping in foil for storage. |

## Notes

Original family note: the bread gets better after it marinates about 2 days. The loaves can be wrapped and stored in the freezer for later, if desired.

Original family note also says this is a very unique dessert bread. It was met with skepticism the first time, but turned out to be quite delicious.

The original page lists prep time as 1 hour 15 minutes, but the method includes about 1 hour of baking. This file separates estimated hands-on prep time from the documented bake time so the dashboard is easier to use.

Servings are estimated from the loaf yield. The original recipe page left servings blank.
