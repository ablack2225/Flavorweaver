## Measured Ingredients

### Sweet Corn Pudding

| Amount | Ingredient | Notes |
|---:|---|---|
| 1 (12 oz) package | Super sweet corn | Prepared according to package directions and cooled |
| 1 cup | 2% milk |  |
| 2 | Eggs | Lightly beaten |
| 3 Tbsp | Mayonnaise |  |
| 3 Tbsp | Spread | Melted |
| 2 Tbsp | Sugar |  |
| 1 Tbsp | All-purpose flour |  |
| Pinch | Salt |  |

## Instructions

| Step | Action |
|---:|---|
| 1 | Preheat oven to 350°F. |
| 2 | Prepare corn according to package directions, then set aside to cool. |
| 3 | Combine remaining ingredients in a large bowl with a wire whisk. |
| 4 | Stir in corn. |
| 5 | Turn mixture into a greased 1-quart shallow casserole. |
| 6 | Bake for 1 hour, or until golden. |
