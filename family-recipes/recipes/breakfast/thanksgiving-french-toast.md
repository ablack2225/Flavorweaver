## Measured Ingredients

### Thanksgiving French Toast

| Amount | Ingredient | Notes |
|---:|---|---|
| 1/2 cup | Margarine | Melted in pan |
| 1 1/2 cups | Brown sugar |  |
| 1 tsp | Cinnamon |  |
| 8-12 slices | Bread |  |
| 8-9 | Eggs |  |
| Pinch | Salt |  |
| 1 3/4-2 cups | Milk |  |

## Instructions

| Step | Action |
|---:|---|
| 1 | Melt margarine in a 9x13 pan. |
| 2 | Add brown sugar and cinnamon and stir. |
| 3 | Layer bread over the margarine, brown sugar, and cinnamon mixture. |
| 4 | Beat eggs and salt together, then add milk. |
| 5 | Pour egg mixture over bread. |
| 6 | Cover and refrigerate overnight. |
| 7 | The next morning, uncover and bake at 350°F for 45 minutes. |
| 8 | Cut into squares and place upside down on plate so the melted sugar-cinnamon mixture is on top. |
