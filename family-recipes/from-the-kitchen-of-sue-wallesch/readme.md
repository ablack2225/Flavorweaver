# From the Kitchen of Sue Wallesch

This collection preserves recipes from Sue Wallesch's family cookbook and kitchen legacy.

Original wording, measurements, titles, and notes should be preserved whenever possible.

Adapted or Amyfied versions should be added as separate recipe files or clearly marked sections so the original recipe remains intact.

## Planned Contents

- `recipes/` — Individual recipes from the Sue Wallesch cookbook collection.
- `original-index.md` — Index of recipes from the original cookbook.
- `cookbook-notes.md` — Notes about the cookbook, family context, formatting decisions, and preservation choices.
