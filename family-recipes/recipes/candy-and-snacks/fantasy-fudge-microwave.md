# Fantasy Fudge (Microwave)

> A preserved family microwave fudge made with chocolate chips, marshmallow cream, evaporated milk, sugar, margarine, vanilla, and chopped nuts.

## Measured Ingredients

### Fantasy Fudge (Microwave)

| Amount | Ingredient | Notes |
|---:|---|---|
| 3/4 cup | Margarine | Melt first |
| 3 cups | Sugar |  |
| 2/3 cup | Evaporated milk |  |
| 1 (12 oz) package | Semi-sweet chocolate chips |  |
| 1 (7 oz) jar | Marshmallow cream |  |
| 1 cup | Chopped nuts |  |
| 1 tsp | Vanilla |  |

## Instructions

| Step | Action |
|---:|---|
| 1 | Melt margarine in a large microwavable bowl for 1 minute. |
| 2 | Add sugar and evaporated milk. |
| 3 | Microwave for 5 minutes, stirring after 3 minutes and again at 5 minutes. |
| 4 | Microwave an additional 5 1/2 minutes, again stirring after 3 minutes. |
| 5 | Remove from microwave and add chocolate chips. Stir until melted. |
| 6 | Blend in marshmallow cream and vanilla. |
| 7 | Stir in chopped nuts. |
| 8 | Pour into a greased 13x9-inch pan. |
| 9 | Cool completely before cutting. |
| 10 | Store in an airtight container. |
