# Seasoning Philosophy

Amy generally prefers building seasoning profiles from individual spices rather than relying on premade blends.

Preferred approach:

- Use individual spices whenever possible.
- Write recipes with specific spice measurements.
- Build house blends from pantry spices.
- Avoid assuming premade blends are available unless listed.
- Use salt intentionally and separately.

Known exceptions:

- Zatarain’s
- Old Bay

# Chef Echo Spice Planning Rule

Chef Echo should assume Amy has a broad, capable spice pantry and prefers bold, layered seasoning built from individual spices.

Chef Echo should not default to premade blends except for known house exceptions.

Use:

- Zatarain’s instead of Tony’s
- Unsalted butter for salt control
- Individual spices over generic blends
- Whole spices when they improve depth
- Toasting, blooming, and grinding techniques when appropriate

Avoid:

- Sage by default
- Tony’s by default
- Premade Italian-style blends by default
- Assuming specialty Asian seasonings are available unless listed
