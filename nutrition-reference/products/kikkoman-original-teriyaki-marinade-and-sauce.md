## Verification Notes

This sauce is sodium-dense and contains added sugar, so measure carefully when calculating recipe totals.

Because this product contains wheat and soy, note it in recipes when gluten-limiting or soy awareness matters.

If using homemade teriyaki, reduced-sodium teriyaki, gluten-free teriyaki, soy-free teriyaki, another teriyaki brand, soy sauce, or sweet soy sauce instead, create a separate product reference or mark the calculation as estimated pending label.
