# Apple Cobbler

> A preserved recipe clipping for apple cobbler with Granny Smith apple filling and a cinnamon-oat topping.

## Ingredient List

| Component | Ingredients |
|---|---|
| Topping | Softened spread, sugar, egg, vanilla, flour, old-fashioned rolled oats, cinnamon |

## Measured Ingredients

### Topping

| Amount | Ingredient | Notes |
|---:|---|---|
| 1/2 cup / 1 stick | Imperial spread | Softened |
| 1 cup | Sugar |  |
| 1 | Egg |  |
| 1 tsp | Vanilla extract |  |
| 1 cup | Flour |  |
| 1 cup | Old-fashioned rolled oats |  |
| 3/4 tsp | Cinnamon |  |

### Apple Filling

| Amount | Ingredient | Notes |
|---:|---|---|
| 8 large | Granny Smith apples | Peeled, cored, and sliced |
| 3/4 cup | Sugar |  |
| 1 1/2 Tbsp | Cornstarch |  |
| 1 1/2 tsp | Cinnamon |  |
| 1/4 tsp | Allspice |  |

## Instructions

| Step | Action |
|---:|---|
| 1 | Preheat oven to 350°F. |
| 2 | Using an electric mixer on medium speed, beat softened spread with sugar in a large bowl until creamy. |
| 3 | Blend in egg and vanilla. |
| 4 | Stir in flour, oats, and cinnamon thoroughly. |
| 5 | In a separate bowl, combine sliced apples, sugar, cornstarch, cinnamon, and allspice. Mix well. |
| 6 | Evenly spoon apple mixture into a shallow 3-quart casserole dish. |
| 7 | Spoon topping mixture evenly over apple filling. |
| 8 | Bake for 1 hour, or until golden brown. |
| 9 | Serve warm or at room temperature. |
