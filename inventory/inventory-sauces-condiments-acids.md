> Chef Echo should use this as a default reference when creating recipes, substitutions, grocery lists, and Flavorweaver meal plans.

---

# Pantry Philosophy

The Amy & Don Kitchen favors bold, layered flavor built from sauces, acids, aromatics, spice, and balance.

Preferred approach:

- Use sauces intentionally.
- Balance richness with acid, heat, herbs, vegetables, or pickles.
- Avoid assuming specialty condiments are available unless listed.
- Build house versions when practical.
- Season rice vinegar, sauces, and dressings manually when better control is needed.
- Use unsalted butter and salt separately for better salt control.

Butter is acceptable.

Heavy dairy should be minimized, optional, or separated for Amy’s portions when practical.

Use caution with:

- Cheese-heavy toppings

- Heavy cream
- Milk-based sauces
- Cheese-heavy breads or shells

Preferred recipe design:

- Butter is fine.
- Heavy cheese should usually be optional or Don-specific.

- Amy’s portion should preserve the core flavor architecture without relying on heavy dairy.

# Chef Echo Sauce & Condiment Planning Rule

Chef Echo should assume Amy and Don keep a strong sauce, vinegar, pickle, and condiment pantry.

When building recipes:

- Use existing sauces before adding specialty items.
- Prefer rice vinegar, apple cider vinegar, white vinegar, lemon, or lime before assuming wine vinegars.
- Use stone-ground mustard when Dijon is unavailable.
- Use Zatarain’s instead of Tony’s for Cajun-style seasoning.
- Treat Red Boat fish sauce as strong.
- Make seasoned rice vinegar manually when needed.
- Make pickled red onions as needed.
- Avoid building Amy’s portion around heavy dairy condiments.
- Add specialty condiments only when they meaningfully support a Flavorweaver project.
